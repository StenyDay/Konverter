Документация к конвертеру величин

Общее функциональное описание:
Конвертер величин – программа представляет из себя отличный конвертор величин. 

Конвертер величин предназначен для перевода физических величин: масса, длина, скорость, объем, площадь и время.
Вы можете практически мгновенно перевести значение между самыми разными единицами измерения.

Запуск программы:
Вы сможете работать двух режимах: 1) консольный режим
                                  2) графический режим

В зависимости от того какой режим Вы выберите, запустите соответствующий файл 
(для консольного режима Konverter.exe, для графического режима Konverter.exe).
//надо бы их назвать по-разному

При запуске Вам предлагается выбрать систему измерения, в которой Вы будете работать. 

Затем выбираете исходную единицу измерения.

Выбираете конечную единицу измерения.
 
Вводите исходные данные в окно.
 
После этого программа выводит конвертированное число.
 
Входные данные – это числа!!!

Чтобы посчитать дробное число, нужно вводить его через запятую (в десятичном виде)!!!

Если во входные данные вводить число в буквенном виде (прописью), программа выведет ошибку.

Для устранения данной ошибки для консольного режима нужно закрыть программу и запустить ее заново, 
при этом вводя во входную строку число.

Для устранения ошибки для графического режима можно:
1) Нажать «Продолжить», во входную строку ввести ЧИСЛО и  посчитать заново.
2) Нажать «Выход» и запустить программу заново, при этом во входную строку вводить число.
